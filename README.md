# Fix-My-Mood-App

React App For Generating Songs From Soundcloud Based On Your Mood.
------------
### Setup Instructions
```
Run $ npm install inside root folder
Run $ npm start
Enjoy :D
```
### Testing Instructions
`Run $ npm test`

### Technologies used:
- React
- react-spinjs
- Axios
- Soundcloud Api
- Jest
- Enzyme
- React Router v4
